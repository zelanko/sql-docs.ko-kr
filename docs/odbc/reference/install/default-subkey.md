레지스트리 기본 하위 키를 사용 하 여 기본 데이터 원본을 지정할 수 있습니다. 이 하위 키는 데이터 소스 사양 하위 키의 특별 한 경우 있고 다른 데이터 소스 사양 하위 키와 동일한 값입니다. 유일한 차이점은 ODBC 데이터 원본 하위 키 아래 값으로 나열 되지 됩니다.
