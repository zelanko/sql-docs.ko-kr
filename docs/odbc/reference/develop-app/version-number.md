# <a name="version-number"></a>버전 번호
다양 한 기능을 사용 하 여 각 ODBC의 여러 버전이 있습니다. 응용 프로그램이 ODBC 드라이버 관리자는 특정 드라이버 지원 호출 하 여 결정 **SQLGetInfo** SQL_ODBC_VER 및 SQL_DRIVER_ODBC_VER 옵션을 사용 하 여 합니다.
