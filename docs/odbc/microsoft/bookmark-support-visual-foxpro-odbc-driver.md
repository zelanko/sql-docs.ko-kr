# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>책갈피 지원(Visual FoxPro ODBC 드라이버)
Visual FoxPro ODBC 드라이버는 간단한 책갈피를 지원합니다. 호출 하는 경우 [SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) SQL_BOOKMARK_PERSISTENCE를 사용 하 여 *정보 항목*, 반환 값은 SQL_BP_SCROLL 합니다.  
  
 책갈피에 대 한 자세한 내용은 참조 하세요. [책갈피 (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md)합니다.
