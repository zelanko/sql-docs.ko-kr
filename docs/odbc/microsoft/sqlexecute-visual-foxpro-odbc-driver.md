# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에서는 Visual FoxPro ODBC 드라이버 관련 정보를 포함합니다. 이 함수에 대 한 일반 정보에서 해당 항목을 참조 하세요 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)합니다.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 준비 된 SQL 문을 실행 (에서 이미 준비 된 문을 [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)). 드라이버 문에 매개 변수가 존재 하는 경우 매개 변수 표식 변수의 현재 값을 사용 합니다.  
  
 자세한 내용은 [SQLExecute](../../odbc/reference/syntax/sqlexecute-function.md) 에 *ODBC 프로그래머 참조*합니다.
