# <a name="sqlconnect-visual-foxpro-odbc-driver"></a>SQLConnect(Visual FoxPro ODBC 드라이버)
> [!NOTE]  
>  이 항목에서는 Visual FoxPro ODBC 드라이버 관련 정보를 포함합니다. 이 함수에 대 한 일반 정보에서 해당 항목을 참조 하세요 [ODBC API 참조](../../odbc/reference/syntax/odbc-api-reference.md)합니다.  
  
 지원: 전체  
  
 ODBC API 규칙: 코어 수준  
  
 일 수 있는 데이터 원본에 연결 된 [데이터베이스](../../odbc/microsoft/visual-foxpro-terminology.md) 의 디렉터리나 [테이블](../../odbc/microsoft/visual-foxpro-terminology.md)합니다. Visual FoxPro ODBC 드라이버는 무시 합니다 *szUID*를 *cbUID*를 *szAuthStr*, 및 *cbAuthStr* 인수입니다.  
  
 자세한 내용은 [SQLConnect](../../odbc/reference/syntax/sqlconnect-function.md) 에 *ODBC 프로그래머 참조*합니다.
