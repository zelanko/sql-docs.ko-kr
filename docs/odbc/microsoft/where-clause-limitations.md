# <a name="where-clause-limitations"></a>WHERE 절 제한 사항
절을 WHERE 절은 40의 최대 수입니다.  
  
 LONGVARCHAR 및 LONGVARBINARY 열 길이 최대 255 자 리터럴에 비교할 수 있지만 매개 변수를 사용 하 여 비교할 수 없습니다.
