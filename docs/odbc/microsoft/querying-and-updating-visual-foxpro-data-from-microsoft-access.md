# <a name="querying-and-updating-visual-foxpro-data-from-microsoft-access"></a>Microsoft Access에서 Visual FoxPro 데이터 쿼리 및 업데이트
쿼리할 수 있으며 링크 테이블 옵션을 사용 하 여 Microsoft Access 데이터베이스에서 Visual FoxPro 데이터베이스에 저장 된 데이터를 업데이트할 수 있습니다.  
  
### <a name="to-link-a-visual-foxpro-database-to-a-microsoft-access-database"></a>Visual FoxPro 데이터베이스 Microsoft Access 데이터베이스에 연결 하려면  
  
1.  Microsoft Access 데이터베이스를 엽니다.  
  
2.  테이블 탭에서 새로 만들기를 클릭 합니다.  
  
3.  새 테이블 대화 상자에서 링크 테이블을 선택 하 고 확인을 클릭 합니다.  
  
4.  연결 대화 상자에서 파일 형식 목록에서에서 ODBC 데이터베이스를 선택 합니다.  
  
5.  SQL 데이터 원본 대화 상자에서 클릭 하 고 쿼리 하려는 Visual FoxPro 데이터에 연결 하는 데이터 소스를 선택 합니다.  
  
6.  테이블 연결 대화 상자에서 클릭 하 고 업데이트 하 고 쿼리 하려는 테이블을 선택 합니다. 연결 된 Visual FoxPro 테이블 Microsoft Access 데이터베이스의 테이블 탭에 표시 됩니다.  
  
 이제 쿼리 및 연결 된 Visual FoxPro 테이블의 데이터를 업데이트 하려면 Microsoft Access를 사용할 수 있습니다. 연결 된 데이터를 변경한 Visual FoxPro 데이터 원본에 다시 전송 됩니다.  
  
 Visual FoxPro 데이터 원본에서 데이터는 영향을 참조 하세요. Microsoft Access에서 수행한 변경 하지 않으려면 [Microsoft Access로 Visual FoxPro 데이터 가져오기](../../odbc/microsoft/importing-visual-foxpro-data-into-microsoft-access.md)합니다.
