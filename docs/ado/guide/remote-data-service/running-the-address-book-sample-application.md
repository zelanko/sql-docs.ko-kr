# <a name="running-the-address-book-sample-application"></a>주소록 예제 애플리케이션 실행
> [!IMPORTANT]
>  Windows 8 및 Windows Server 2012 부터는 RDS 서버 구성 요소는 더 이상 포함 된 Windows 운영 체제에서 (Windows 8을 참조 하 고 [Windows Server 2012 호환성 설명서](https://www.microsoft.com/download/details.aspx?id=27416) 자세한). RDS 클라이언트 구성 요소는 Windows의 이후 버전에서 제거 됩니다. 새 개발 작업에서는 이 기능을 사용하지 않도록 하고, 현재 이 기능을 사용하는 애플리케이션은 수정하세요. RDS를 사용 하는 응용 프로그램을 마이그레이션해야 [WCF 데이터 서비스](https://go.microsoft.com/fwlink/?LinkId=199565)합니다.  
  
 주소록 응용 프로그램을 실행 하려면이 절차를 따릅니다.

### <a name="to-run-this-application"></a>이 응용 프로그램을 실행 하려면  
  
1.  Microsoft SQL Server 실행 되 고 있는지 확인 하십시오. 클릭 **시작**, 가리킨 **프로그램**를 가리킨 **Microsoft SQL Server 7.0**를 클릭 하 고 **Service Manager**. 흰색 원에 녹색 화살표가 있으면 SQL Server가 실행 됩니다. 없는 경우 (됩니다 빨간색 사각형이 흰색 원 안에)를 클릭 **시작/계속**합니다.  
  
2.  Microsoft Internet Explorer 4.0 이상에서는 다음 주소를 입력 합니다.  
  
     **https://** *webserver* **/RDS/AddressBook/AddrBook.asp**  
  
     여기서 *webserver* RDS 서버 구성 요소를 설치할 웹 서버의 이름입니다.  
  
3.  다음 제목이 "프로그램 관리자," 모든 사용자를 나열가 자신의 전자 메일 이름에 따라 사용자를 검색 하거나 기존 레코드를 편집 같은 주소록 예제 응용 프로그램에서는 다양 한 시나리오를 시도할 수 있습니다. 클릭 **찾을** 사용 가능한 모든 이름으로 데이터 눈금을 채웁니다.
