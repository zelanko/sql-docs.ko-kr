합니다 [데이터 가져오기](../../../ado/guide/data/getting-data.md) 섹션에는 하나 이상의 데이터 원본에서 데이터를 검색 하는 방법을 설명 했습니다 **Recordset** 개체입니다. 이 섹션에서는 설명 **레코드 집합** 자세히 탐색 하는 방법 등을 **레코드 집합** 해당 데이터를 봅니다.
