# <a name="precision-property-adox"></a>전체 자릿수 속성 (ADOX)
에 있는 데이터 값의 최대 전체 자릿수를 나타내는 [열](../../../ado/reference/adox-api/column-object-adox.md)합니다.  
  
## <a name="settings-and-return-values"></a>설정 및 반환 값  
 설정 하 고 반환는 **긴** 열에 있는 데이터 값의 최대 전체 자릿수 값 때는 [형식](../../../ado/reference/adox-api/type-property-column-adox.md) 속성은 숫자 유형입니다. **전체 자릿수** 다른 모든 데이터 형식에 대해 무시 됩니다.  
  
## <a name="remarks"></a>주의  
 기본값은 0 (**0**).  
  
 이 속성에 대 한 읽기 전용은 [열](../../../ado/reference/adox-api/column-object-adox.md) 컬렉션에 이미 추가 된 개체입니다.
