# <a name="localdberrorcallerisnotowner"></a>LOCALDB_ERROR_CALLER_IS_NOT_OWNER
    
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|282|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|API 호출자는 로컬 데이터베이스 인스턴스 소유자가 아닙니다.|  
  
## <a name="explanation"></a>설명  
 요청한 작업을 실행하려면 인스턴스 소유자여야 합니다.  
  
## <a name="user-action"></a>사용자 동작  
 인스턴스 소유자에게 문의하여 도움을 받으십시오.
