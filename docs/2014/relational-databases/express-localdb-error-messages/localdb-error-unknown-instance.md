# <a name="localdberrorunknowninstance"></a>LOCALDB_ERROR_UNKNOWN_INSTANCE
    
## <a name="details"></a>설명  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|263|  
|이벤트 원본|SQL Server 로컬 데이터베이스 런타임 12.0|  
|구성 요소|로컬 데이터베이스 런타임 API|  
|메시지 텍스트|지정한 로컬 데이터베이스 인스턴스가 없습니다.|  
  
## <a name="explanation"></a>설명  
 지정한 인스턴스가 존재하지 않습니다.  
  
## <a name="user-action"></a>사용자 동작  
 지정한 인스턴스 이름이 올바른지 확인하십시오.
