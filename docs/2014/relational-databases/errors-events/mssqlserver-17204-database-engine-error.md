|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|17204|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBLKIO_DEVOPENFAILED|
