# <a name="copy-items-in-a-solution"></a>솔루션에서 항목 복사
  솔루션 탐색기나 Windows 탐색기를 사용하여 항목을 복사할 수 있습니다.  
  
### <a name="to-copy-items-within-solution-explorer"></a>솔루션 탐색기에서 항목을 복사하려면  
  
1.  솔루션 탐색기에서 복사할 항목을 선택합니다.  
  
2.  **편집** 메뉴에서 **복사**를 클릭합니다.  
  
3.  솔루션 탐색기에서 대상 프로젝트를 선택합니다.  
  
4.  **편집** 메뉴에서 **붙여넣기**를 클릭합니다.  
  
> [!NOTE]  
>  프로젝트 간의 연결은 복사할 수 없습니다. 연관된 연결이 있는 쿼리를 다른 프로젝트에 복사하면 연관된 연결은 대상 프로젝트에 복사되지 않습니다. 따라서 대상 프로젝트에서 연결을 수동으로 만들어야 합니다.
