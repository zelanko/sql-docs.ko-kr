# <a name="browse-for-objects-object-explorer"></a>개체 찾아보기(개체 탐색기)
  선택한 유형의 개체 목록을 표시합니다. 해당 확인란을 선택하여 이 목록에서 하나 이상의 개체를 선택합니다. **개체 찾아보기** 상자에는 동작에 해당하는 항목만 표시되며 보기 권한이 있는 항목만 표시됩니다.  
  
## <a name="options"></a>변수  
 **확인란**  
 개체를 포함하려면 선택합니다.  
  
 **아이콘**  
 개체 유형을 나타내는 아이콘입니다.  
  
 **이름**  
 개체 이름입니다.  
  
 **형식**  
 데이터베이스 개체의 유형입니다.
