# <a name="select-existing-dimensions-cube-wizard"></a>기존 차원 선택(큐브 마법사)
  **기존 차원 선택** 페이지를 사용하여 큐브에 포함할 기존 차원을 선택할 수 있습니다.  
  
## <a name="options"></a>변수  
 **Dimensions**  
 큐브에 포함할 차원을 선택합니다.  
  
 이 마법사의 **측정값 그룹 테이블 선택** 페이지에서 선택한 측정값 그룹에서 사용할 수 있는 차원만 목록에 표시됩니다.  
  
 기본적으로 모든 차원이 선택되어 있습니다.
