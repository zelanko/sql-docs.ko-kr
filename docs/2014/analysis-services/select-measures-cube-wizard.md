# <a name="select-measures-cube-wizard"></a>측정값 선택(큐브 마법사)
  **측정값 선택** 페이지를 사용하여 만들 측정값 그룹과 측정값을 선택할 수 있습니다.  
  
## <a name="options"></a>변수  
 **측정값**  
 사용할 수 있는 측정값 그룹과 해당 측정값을 표시합니다.  
  
 측정값 그룹 또는 측정값이 생성되도록 표시하려면 해당 확인란을 선택합니다. 기본적으로 모든 측정값 그룹과 측정값이 선택되어 있습니다.  
  
 측정값 그룹 또는 측정값의 이름을 변경하려면 해당 이름을 클릭하고 새 이름을 입력합니다.
