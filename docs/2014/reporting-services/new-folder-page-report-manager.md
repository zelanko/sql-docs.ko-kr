# <a name="new-folder-page-report-manager"></a>새 폴더 페이지(보고서 관리자)
  새 폴더 페이지를 사용하여 보고서 서버 폴더 계층에 새 폴더를 만들 수 있습니다. 이 페이지에서 만드는 폴더는 보고서 서버 데이터베이스에 저장되는 가상 폴더입니다. 컴퓨터의 파일 시스템에서는 폴더가 생성되지 않습니다.  
  
 폴더는 현재 위치에서 현재 선택된 폴더의 포함된 폴더로 만들어집니다. 폴더를 만들려면 우선 폴더를 만들 위치로 이동해야 합니다.  
  
 폴더를 만든 후 폴더의 일반 속성 페이지에서 폴더 이름 및 설명을 수정할 수 있습니다.  
  
## <a name="navigation"></a>탐색  
 사용자 인터페이스(UI)에서 이 위치를 탐색하려면 다음 절차를 사용하십시오.  
  
###### <a name="to-open-the-new-folder-page"></a>새 폴더 페이지를 열려면  
  
1.  보고서 관리자를 열고 새 폴더를 만들려는 폴더로 이동합니다.  
  
2.  도구 모음에서 **새 폴더**를 클릭합니다.  
  
## <a name="options"></a>변수  
 **이름**  
 폴더 이름을 지정합니다. 이름은 하나 이상의 영숫자 문자를 포함해야 합니다. 공백과 특정 기호도 포함할 수 있습니다. 이름을 지정할 때 ; ? : @ & = +, $ / * \< > | "또는 / 이름을 지정 합니다.  
  
 **설명**  
 폴더 내용에 대한 설명을 입력합니다. 이 설명은 해당 폴더 액세스 권한이 있는 사용자의 내용 페이지에 표시됩니다.
