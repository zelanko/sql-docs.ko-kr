# <a name="report-f1-help"></a>보고서 F1 도움말
  이 절의 항목에서는 **보고서 속성** 대화 상자에서 보고서에 설정할 수 있는 옵션에 대해 설명합니다. **보고서 속성** 대화 상자를 열려면 보고서 디자이너의 배경을 마우스 오른쪽 단추로 클릭한 다음 **보고서 속성**을 클릭합니다.  
  
> [!NOTE]  
>  보고서 본문을 보고서로 착각하기가 쉽습니다. 보고서 본문은 보고서의 내용을 포함하는 보고서 요소입니다. 보고서는 보고서 본문 및 선택적으로 보고서 머리글과 바닥글로 구성됩니다. 보고서 본문을 마우스 오른쪽 단추로 클릭하는 경우 보고서 속성을 설정하거나 액세스할 수 없습니다.  
  
 이 항목에 액세스하려면 **보고서 속성** 대화 상자에서 **도움말**을 클릭합니다.
