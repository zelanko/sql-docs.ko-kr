# <a name="report-data-f1-help"></a>보고서 데이터 F1 도움말
  이 절의 항목에서는 보고서 디자이너에서 사용할 수 있는 쿼리 디자이너에 대한 정보뿐만 아니라 데이터 원본, 공유 데이터 원본 및 데이터 집합에 설정할 수 있는 옵션에 대해 설명합니다.  
  
 이 항목에 액세스하려면 **보고서 속성** 대화 상자에서 **도움말**을 클릭합니다.
