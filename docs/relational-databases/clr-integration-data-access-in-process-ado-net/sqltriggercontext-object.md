# <a name="sqltriggercontext-object"></a>SqlTriggerContext 개체
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  **SqlTriggerContext** 클래스는 트리거에 대한 컨텍스트 정보를 제공합니다. 이 컨텍스트 정보에는 트리거를 발생시킨 동작 유형, UPDATE 작업에서 수정된 열, DDL(데이터 정의 언어) 트리거의 경우 트리거 작업을 설명하는 **EventData** 구조가 포함됩니다. 자세한 내용 및 사용 하는 방법의 예는 **SqlTriggerContext** 클래스를 참조 하십시오 [CLR 트리거](https://msdn.microsoft.com/library/302a4e4a-3172-42b6-9cc0-4a971ab49c1c)합니다.  
  
 자세한 내용은 .NET Framework SDK 설명서의 **Microsoft.SqlServer.Server.SqlTriggerContext** 클래스 참조 설명서를 참조하십시오.
