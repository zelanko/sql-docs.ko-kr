  [!INCLUDE[csprcs](../../includes/csprcs-md.md)].NET은 .Net의 CLR (공용 언어 런타임)에서 지 원하는 관리 되는 프로그래밍 언어 중 하나입니다 [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] .
