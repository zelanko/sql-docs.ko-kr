# <a name="stored-procedure-properties-general-page"></a>저장 프로시저 속성(일반 페이지)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
  저장 프로시저에 대한 정보를 표시합니다. 모든 정보는 읽기 전용입니다.  
  
## <a name="options"></a>옵션  
 **Database**  
 이 저장 프로시저를 포함하는 데이터베이스의 이름입니다.  
  
 **Server**  
 현재 서버 인스턴스의 이름입니다.  
  
 **사용자**  
 이 연결을 사용하는 사용자의 이름입니다.  
  
 **만든 날짜**  
 저장 프로시저를 만든 날짜를 표시합니다.  
  
 **이름**  
 현재 저장 프로시저의 이름입니다.  
  
 **스키마**  
 저장 프로시저를 소유하는 스키마를 표시합니다.  
  
 **시스템 개체**  
 저장 프로시저가 시스템 개체인지 여부를 나타냅니다. 사용 가능한 값은 True와 False입니다.  
  
 **ANSI NULL**  
 개체가 ANSI NULL 옵션으로 생성되었는지 여부를 나타냅니다.  
  
 **암호화됨**  
 저장 프로시저를 암호화하는지 여부를 나타냅니다. 사용 가능한 값은 True와 False입니다.  
  
 **복제용**  
 개체가 FOR REPLICATION 옵션으로 생성되었는지 여부를 나타냅니다.  
  
 **따옴표 붙은 식별자**  
 개체가 따옴표 붙은 식별자 옵션으로 생성되었는지 여부를 나타냅니다.  
  
 **다시 컴파일**  
 개체가 RECOMPILE 옵션으로 생성되었는지 여부를 나타냅니다.
