### <a name="internal-dmvs"></a>내부 Dmv

내부 용도로 및 문서가 없는 직접 제공 하는 추가 Dmv 있습니다. 메모리 최적화 테이블의 영역에서 문서화 되지 않은 Dmv는 다음과 같습니다.

- sys.dm_xtp_threads
- sys.dm_xtp_transaction_recent_rows
