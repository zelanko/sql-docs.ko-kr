## <a name="details"></a>세부 정보  
  
|||  
|-|-|  
|제품 이름|SQL Server|  
|이벤트 ID|7914|  
|이벤트 원본|MSSQLSERVER|  
|구성 요소|SQLEngine|  
|심볼 이름|DBCC2_REPAIR_ALLOCATION_PAGE_REBUILT|  
|메시지 텍스트|복구: P_ID에서 PAGE_TYPE 페이지가 다시 작성되었습니다.|  
  
## <a name="explanation"></a>설명  
GAM 또는 SGAM 페이지가 PFS 페이지 데이터를 사용하여 다시 작성되었음을 나타내는 REPAIR의 정보 페이지입니다.
