# <a name="objects-event-category"></a>Objects 이벤트 범주
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)] **Objects** 이벤트 범주에는 일반적인 개체 이벤트가 포함됩니다.  
  
## <a name="in-this-section"></a>섹션 내용  
  
|항목|설명|  
|-----------|-----------------|  
|[Auto Stats 이벤트 클래스](../../relational-databases/event-classes/auto-stats-event-class.md)|인덱스 및 열 통계가 자동으로 업데이트되었음을 나타냅니다.|  
|[Object:Altered 이벤트 클래스](../../relational-databases/event-classes/object-altered-event-class.md)|ALTER DATABASE 문이나 ALTER TABLE 문 등에 의해 개체가 변경되었음을 나타냅니다.|  
|[Object:Created 이벤트 클래스](../../relational-databases/event-classes/object-created-event-class.md)|CREATE INDEX 문, CREATE TABLE 문 또는 CREATE DATABASE 문 등에 의해 개체가 생성되었음을 나타냅니다.|  
|[Object:Deleted 이벤트 클래스](../../relational-databases/event-classes/object-deleted-event-class.md)|DROP INDEX 문이나 DROP TABLE 문 등에 의해 개체가 삭제되었음을 나타냅니다.|
