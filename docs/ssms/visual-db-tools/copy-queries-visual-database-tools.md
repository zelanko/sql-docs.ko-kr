# <a name="copy-queries-visual-database-tools"></a>쿼리 복사(Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] 쿼리 본문을 새 쿼리로 복사하여 새로운 쿼리를 만들기 위한 기반으로 사용할 수 있습니다.  
  
### <a name="to-copy-a-query"></a>쿼리를 복사하려면  
  
1.  **보기** 메뉴에서 솔루션 탐색기를 클릭합니다.  
  
2.  솔루션 탐색기에서 복사하려는 쿼리를 마우스 오른쪽 단추로 클릭하고 바로 가기 메뉴에서 **열기**를 선택합니다.  
  
3.  SQL 창에서 복사하려는 쿼리의 일부를 복사합니다.  
  
4.  복사한 SQL을 붙여넣을 쿼리를 열거나 새 쿼리를 만듭니다.  
  
5.  SQL 코드를 추가할 위치에 커서를 놓습니다.  
  
6.  마우스 오른쪽 단추를 클릭하고 바로 가기 메뉴에서 **붙여넣기**를 클릭합니다.
