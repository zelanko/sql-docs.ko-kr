# <a name="dmx-tutorials-analysis-services---data-mining"></a>DMX 자습서(Analysis Services - 데이터 마이닝)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  다음 자습서에서는 데이터 마이닝 구조 및 모델과 관련하여 DMX(Data Mining Extensions)를 사용하는 방법을 소개합니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [Bike Buyer DMX 자습서](https://msdn.microsoft.com/library/4b634cc1-86dc-42ec-9804-a19292fe8448)  
  
 이 자습서에서는 DMX 쿼리 언어를 사용하여 마이닝 모델을 작성, 학습 및 탐색하는 방법을 배웁니다. 이러한 마이닝 모델을 사용하여 고객이 특정 제품을 구입할지 여부에 대한 예측을 만들 수 있습니다.  
  
 [Market Basket DMX 자습서](https://msdn.microsoft.com/library/6e262a1d-c89e-4033-8368-46cf25168ef5)  
  
 이 자습서에서는 동시에 구입되는 경향이 있는 제품을 예측하는 마이닝 모델을 만드는 방법을 배웁니다. 또한 데이터 마이닝에서 중첩 테이블을 사용하는 방법을 보여 줍니다.
