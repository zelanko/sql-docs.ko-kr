# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>분석 플랫폼 시스템의 관리 콘솔 경고 이해
경고는 어플라이언스 **관리 콘솔과** SCOM ( **System Center Operations Manager** )에 표시 됩니다. 이 경고 목록을 사용 하면 추가 조사가 필요한 경고를 식별할 수 있습니다.  
  
Internet Explorer를 사용 하 여 **관리 콘솔** 에 연결 하는 방법에 대 한 자세한 내용은 [관리 콘솔을 사용 하 여 어플라이언스 모니터링 &#40;분석 플랫폼 시스템&#41;](monitor-the-appliance-by-using-the-admin-console.md)을 참조 하세요. **SCOM** 에 대 한 자세한 내용은 [System Center Operations Manager &#40;분석 플랫폼 시스템을 사용 하 여 어플라이언스 모니터링](monitor-the-appliance-by-using-system-center-operations-manager.md) 을 참조 하세요&#41;  
  
Transact-sql을 사용 하 여 경고 정보를 얻는 방법에 대 한 자세한 내용은 \- [시스템 뷰를 사용 하 여 어플라이언스 모니터링 &#40;분석 플랫폼 시스템&#41;](monitor-the-appliance-by-using-system-views.md)을 참조 하세요.  
  
## <a name="types-of-alerts"></a>경고 유형  
정상 상태를 나타내는 경고 이름은 일반적으로 조사가 필요 하지 않습니다. NON_CRITICAL 단어가 포함 된 경고 이름에는 때때로 조치가 필요 합니다. 다른 모든 유형의 경고에 대 한 조사가 필요 합니다.
