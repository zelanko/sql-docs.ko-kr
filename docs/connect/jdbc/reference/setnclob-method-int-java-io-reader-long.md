# <a name="setnclob-method-int-javaioreader-long"></a>setNClob 메서드(int, java.io.Reader, long)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  지정된 매개 변수를 지정된 문자 길이의 지정된 Reader 개체로 설정합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
public final void setNClob(int parameterIndex,  
                    java.io.Reader reader,  
                    long length)  
```  
  
#### <a name="parameters"></a>매개 변수  
 *된*  
  
 매개 변수 인덱스를 나타내는 **int**입니다.  
  
 *reader*  
  
 매개 변수 값을 나타내는 Reader 개체입니다.  
  
 *length*  
  
 매개 변수 값의 문자 수를 나타내는 **long**입니다.  
  
## <a name="exceptions"></a>예외  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 이 setNClob 메서드는 java.sql.PreparedStatement 인터페이스의 setNClob 메서드에 의해 지정됩니다.
