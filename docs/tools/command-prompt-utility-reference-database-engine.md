## <a name="command-prompt-utilities-syntax-conventions"></a>명령 프롬프트 유틸리티 구문 규칙  
  
|**규칙**|**용도**|  
|--------------------|------------------|  
|대문자|운영 체제 수준에서 사용하는 문 및 용어|  
|`monospace`|예제 명령 및 프로그램 코드|  
|*기울임꼴*|사용자가 제공하는 매개 변수|  
|**굵게**|표시된 대로 입력해야 하는 명령, 매개 변수 및 다른 구문|
